# قاموسي المدرسي

تطبيق Flutter Android لقاموس إنجليزي عربي للمنهاج الأردني من الروضة إلى الصف الثامن.

## الميزات
- 900 كلمة مبدئية: 100 كلمة لكل صف بدون تكرار.
- قاموس وبطاقات واختبارات اختيار من متعدد.
- نطق الكلمات الإنجليزية عبر TTS.
- 3 ملفات شخصية كحد أقصى.
- نقاط: 0.5 لكل إجابة صحيحة.
- كلمات صعبة تلقائياً.
- إنجازات ونجوم.
- استيراد JSON وExcel.
- مكان إعلان علوي وسفلي جاهز بدون تفعيل.

## طريقة الرفع
1. افتح GitHub وأنشئ مستودعاً جديداً.
2. ارفع كل ملفات المشروع كما هي.
3. ادخل تبويب Actions.
4. شغّل Build APK Direct.
5. بعد النجاح سيظهر الملف:
   `apk/qamoosi-school.apk`

## ملف Excel للاستيراد
الأعمدة المطلوبة بالترتيب:
`grade`, `word_en`, `meaning_ar`

مثال:
`KG, apple, تفاحة`
